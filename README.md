Repositori ini berisi proyek automasi pengujian untuk website OrangeHRM Demo. Proyek ini dikembangkan sebagai bagian dari Final Project Quality Assurance Engineer Sanbercode menggunakan Cypress dengan arsitektur Page Object Model (POM).

### Tech Stack & Tools:
* **Framework:** Cypress
* **Editor:** Visual Studio Code
* **Language:** JavaScript
* **Documentation:** Google Spreadsheets
* **Design Pattern:** Page Object Model (POM)
* **Version Control & Repository Host:** Git & GitHub

## Cara Menjalankan Project:
1. Clone repository ini.
2. Buka terminal di folder project.
3. Jalankan perintah: `npm install` (untuk mengunduh library yang diperlukan dan menampilkan folder node_modules lagi).
4. Jalankan Cypress dengan: `npx cypress open` atau `npx cypress run`.

## Catatan:
Untuk folder "node_modules" sengaja saya masukkan ke .gitignore karena ukuran yang terlalu besar.
